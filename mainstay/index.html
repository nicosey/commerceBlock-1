How it works

MainStay - trustless proof of a single, verifiable history of state changes MainStay is a unique attestation service protocol that allows not only for individual point-in-time attestations but it also provides cryptographic proof of one single history of data or system changes.

This 'Proof-of-Immutable-State' is obtained via the trustless immutability inherent in a strong proof-of-work blockchain (eg. Bitcoin). MainStay utilizes a 'Staychain', successive commitments into a sequence of linked transactions to the underlying blockchain which creates a single, non-branching history backed by the proof-of-work of the underlying blockchain.
